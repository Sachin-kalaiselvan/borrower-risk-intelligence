# Dataset

This project uses the Home Credit Default Risk dataset from Kaggle.

The data files are not stored in this repo due to size. To download:

1. Go to [kaggle.com/c/home-credit-default-risk](https://www.kaggle.com/c/home-credit-default-risk)
2. Accept the competition rules
3. Download `application_train.csv`
4. Place it in this folder before running the notebooks

The notebooks are set up to load from Google Drive if running in Colab. See `notebooks/02_feature_engineering.ipynb` for full setup instructions.
